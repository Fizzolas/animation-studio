# Workshop Integration

This directory handles the community workshop system for sharing and distributing plugins.

## Structure

- `client/` - Workshop client for browsing and downloading
- `uploader/` - Tool for uploading plugins to workshop
- `api/` - Workshop API integration
- `cache/` - Local cache of downloaded workshop items

## Workshop Features

### For Users
- Browse community tools and plugins
- One-click install of workshop items
- Automatic updates for subscribed items
- Rating and review system
- Collections and favorites

### For Creators
- Upload custom tools and plugins
- Version management
- Usage statistics
- Community feedback

## Workshop Backend

The workshop backend system (hosting, database, CDN) is separate from this client implementation. Details on the backend architecture will be added as the project develops.
