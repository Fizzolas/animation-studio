# Example Projects

This directory contains example projects and tutorials to help users learn Animation Studio.

## Structure

- `2d_basics/` - Basic 2D drawing and animation examples
- `3d_basics/` - Basic 3D modeling and animation examples
- `workflows/` - Complete workflow examples
- `scripts/` - Example scripts and automation
- `plugins/` - Example plugin implementations

## Example Projects

### 2D Examples
- Simple bouncing ball animation
- Character walk cycle
- Layer-based composition
- SVG import and animation

### 3D Examples
- Basic modeling workflow
- Character rigging basics
- Camera animation
- Lighting and rendering

### Scripting Examples
- Custom tool creation
- Batch operations
- Procedural animation
- Export pipeline automation

### Integration Examples
- Export to game engines
- Integration with other DCC tools
- Workshop plugin development

## File Format

All examples will be provided as:
- `.anst` project files (once format is defined)
- Exported video/image previews
- Step-by-step tutorial documentation
