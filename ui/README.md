# User Interface

This directory contains the UI framework and components for the Animation Studio.

## Structure

- `framework/` - Core UI framework integration (Qt/wxWidgets/custom)
- `panels/` - Dockable panels (viewport, timeline, properties, scripting console)
- `widgets/` - Custom UI widgets and controls
- `themes/` - Soft, comfortable UI themes and styling
- `layouts/` - Layout management and workspace presets
- `dialogs/` - Dialog windows (import/export, preferences, etc.)

## Design Principles

- **Soft, Modern Aesthetic**: Comfortable colors and rounded edges
- **Hierarchical Organization**: Clear folder/layer structure
- **Customizable**: Drag-and-drop panel layouts
- **Searchable**: Quick tool search and command palette
- **Contextual**: Right-click menus based on selection

## Technology

- **Framework**: To be determined (Qt recommended for cross-platform)
- **Styling**: Custom theming system
- **Python Integration**: UI components accessible from Python API
